# Simple Kanban Board

A simple Kanban board built using **HTML, CSS, and vanilla JavaScript** to practice core front-end concepts and native drag-and-drop.

## Features

* Three columns: **To Do**, **In Progress**, **Done**
* Drag cards between columns
* Visual feedback while dragging
* Responsive layout for smaller screens

## Tech Stack

* HTML
* CSS (Flexbox)
* JavaScript (native Drag and Drop API)

## What this project is

* A **learning project**
* Focused on understanding **DOM manipulation** and **drag-and-drop events**
* No frameworks, no libraries

## What this project is NOT

* No card reordering within a column
* No data persistence (refresh resets the board)
* No mobile/touch drag support

## How to Run

1. Clone or download the repository
2. Open `index.html` in a browser
3. Drag cards between columns

## Future Improvements

* Reorder cards within a column
* Save board state using `localStorage`
* Better mobile support
* Add new cards dynamically
